# Rural Health AI Assistant
An AI-powered clinical decision support tool designed for rural healthcare workers.
## Features
- Symptom-based disease prediction
- Probability ranking of diseases
- Urgency detection system
- Streamlit web interface
## Tech Stack
- Python
- Scikit-learn (Random Forest)
- Streamlit
## Future Improvements
- Larger datasets (Kaggle)
- NLP symptom input
- Severity scoring
- Deployment
## Disclaimer
This tool is still in testing phrases and is for educational purposes only and does not replace professional medical advice.
